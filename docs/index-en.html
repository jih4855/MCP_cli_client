MCP CLI Client

Multi-LLM CLI client for leveraging MCP (Model Context Protocol) servers in terminal environments

Project Purpose

Provides CLI-based AI assistant that can utilize various tools by integrating with MCP servers in Linux/Unix environments.

Multi-LLM Support

Support for multiple LLM providers: Gemini, Ollama, OpenAI

Global Install (Recommended)

Recommended installation method for Ubuntu/Linux environments.

Terminal
$ sudo apt update && sudo apt install pipx
$ pipx ensurepath
$ source ~/.bashrc

Clone and Install Project

Clone the project and install with pipx.

Terminal
$ git clone https://github.com/jih4855/MCP_cli_client.git
$ cd MCP_cli_client
$ pipx install .
$ pipx ensurepath
$ source ~/.bashrc

Run Anywhere

Now you can use the mcp-client command anywhere.

Terminal
$ mcp-client
Running safely in isolated environment

Clone Repository

Copy the project to your local machine.

Terminal
$ git clone https://github.com/jih4855/MCP_cli_client.git
$ cd MCP_cli_client

Setup Virtual Environment and Install

Create virtual environment and install as global command.

Terminal
$ python -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
$ pip install -e .

Run Anywhere

Now you can use the mcp-client command anywhere.

Terminal
$ mcp-client
No virtual environment activation needed

Configuration

Environment Variables (.env)

Configure your Gemini API key.

.env
api_key=your_gemini_api_key_here

LLM Configuration (config.yaml)

Set up LLM model and system prompt.

config.yaml
system_prompt: |
You are a helpful AI assistant.
model: gemini-2.5-flash
provider: gemini
api_base: None
max_context: 20
session_id: "default"

MCP Server Configuration (mcp_config.json)

Configure MCP servers to use.

mcp_config.json
{
"mcpServers": {
"ubuntu-info": {
"command": "python",
"args": ["mcp_server/MCP_official.py"]
},
"filesystem": {
"command": "npx",
"args": ["-y", "@modelcontextprotocol/server-filesystem@latest", "/home"]
}
}
}

Usage

Basic Conversation

Start a simple conversation. Type "끝" to exit.

Terminal
user: Hello!
assistant: Hello! How can I help you?
user: 끝
Conversation ended

Tool Usage

MCP server tools are automatically executed in parallel.

Terminal
user: Show me current time and system information
Tool call: get_current_time
Tool call: get_ubuntu_system_info
Tool execution completed
assistant: Current time is Jan 8, 2025, and running on Ubuntu 22.04 system.

Architecture

LLM Class

Multi-provider abstraction

→

MCPmanager

MCP server connection and tool execution management

→

MemoryManager

SQLite conversation storage

Async Processing Flow

  1. Parallel connection with MCP servers
  2. Collect available tools list
  3. LLM decides tool invocation
  4. Execute required tools in parallel
  5. Integrate results for final response
